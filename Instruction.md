# The debuger

Projet de jeu 2D fait avec Maki.

## Prérequis

- Node.js
- npm
- Un navigateur web pour exécuter le jeu localement

## Installation et utilisation

Le paquet `@tialops/maki` est installé localement. Voici les commandes disponibles :

### Commandes principales

```bash
# Lancer le serveur de développement
npm run dev
# ou
npm run maki -- dev


### Installation globale (optionnel)

Pour utiliser les commandes directement, vous pouvez installer globalement :

```bash
npm install -g @tialops/maki
```

## Histoire du jeu

Vous êtes absorbé dans un monde virtuel rempli de bugs. Pour vous en sortir, vous devez réparer ce monde.

### Première quête : régler la caméra

Pour résoudre ce problème, vous devez attraper un chat qui est la source du chaos. Lorsque vous vous approchez du chat, une notification s'affiche : *« Appuyer sur A pour attraper le chat »*. Une jauge apparaît au-dessus du chat et se remplit lentement si le chat reste immobilisé.

**Attention :** Cette mission est particulièrement difficile !

## Contrôles

### Déplacement
- **WASD** ou **Touches directionnelles** : se déplacer

### Interaction
- **E** ou **R** : interagir avec les objets du monde
