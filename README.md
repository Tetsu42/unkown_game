# unkown_game

Projet de jeu 3D en cours de développement avec Maki.


## Requirements
- Node.js
- npm
- A browser for running the local game

## Utilisation

Le paquet `@tialops/maki` est installé en local, donc la commande ne sera pas disponible directement dans le shell. Utilise l'un de ces équivalents :

- `npm exec maki -- new game`
- `npm run maki -- new game`
(ou installation globale `npm install -g @tialops/maki`)

Commandes utiles :

- `npm run dev || npm run maki --dev` pour lancer le serveur de développement 
- `npm run build` pour générer la version de production

- `npx maki tilemap`  création de tilemap (taille, nom)

 histoire du jeu:
    Vous êtes absorbé dans un monde virtuel plein de bug.
    Pour s'en sortir, vous devrez réparer le monde.
    1ère quête : régler la camera, elle devrait fixé le joueur , au lieu de cela un chat en est dévenu la cible.
    Pour résoudre le problème, il faut que vous attrapez le chat. Lorsqu'on approche du chat un notif "Appuyer sur A pour attraper le chat" une jauge s'affichera au dessus du chat (si le chat est immobilisé, le jauge se rempli tout doucement) 
    bémole? c'est un mission difficile 
    Action : vous pouvez intéragir avec les objets, les déplacer,colleter si c'est un colléctible
    moyen rapide pour résoudre la quête : jouer avec l'humeur du chat,(une barre qui se trouve en haut à droite de l'ecran répresente l'humeur du chat)
    du lait, herbe chat, poisson, plus on donne à manger au chat plus il sera de bonne humeur ainsi il ne bougera plus
